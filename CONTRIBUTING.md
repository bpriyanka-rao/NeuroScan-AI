# Contributing

Thank you for your interest in NeuroScan AI.

## How to contribute

1. Fork the repository.
2. Create a new branch with a descriptive name.
3. Submit a pull request with a clear summary of your changes.

## Project standards

- Keep code readable and well-organized.
- Use descriptive variable and function names.
- Add comments for non-obvious logic.
- Keep UI text concise and user-friendly.

## Local development

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app/app.py
```

## Reporting Issues

Please open GitHub issues for bugs, feature requests, or deployment problems.
